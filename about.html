Image to Old Video

Version: 0.2.0  |  Language: Java 25  |  UI: JavaFX

What It Does

Image to Old Video takes a still image and converts it into a short video with a vintage old-film look. Drag in a .png, .jpg, or .bmp image, preview the effect frame-by-frame, and export a ready-to-use MP4 file.

Current Features

  • Image Gallery and Video Gallery tabs for content management.
  • Drag-and-drop image loading and image preview replacement.
  • Per-effect ON/OFF toggles with compact controls for variance, frequency, and randomness.
  • Mini video preview controls (frame slider, step, play/pause, loop, 1-10 second preview length, 30/60 FPS).
  • Configurable video duration (1 to 300 seconds) and output folder.
  • Video Gallery storage settings for max space and max video count, including usage indicator.
  • Generated filenames include source image name, preset name, and timestamp.
  • MP4 output encoded with H.264 via Jcodec.

How to Run

mvn javafx:run

Notes

  • The split divider between preview and effects sections is persisted between launches.
  • The export section is anchored above the status bar.
  • Video playback from gallery uses the OS default video player.
