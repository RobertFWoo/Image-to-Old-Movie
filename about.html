Image to Old Video

Version: 0.1.0  |  Language: Java 21  |  UI: JavaFX

What It Does

Image to Old Video takes a still image and converts it into a short video with a vintage old-film look. Drag in a .png, .jpg, or .bmp image, preview the sepia effect, and export a ready-to-use MP4 file.

Phase 1 Features

  • Drag-and-drop or browse to load an image.
  • Sepia tone filter applied automatically.
  • Live preview of the processed image before export.
  • Configurable video duration (default: 20 seconds).
  • Configurable output folder (defaults to D:\ImageToOldVideo, falls back to C:\).
  • MP4 output encoded with H.264 via Jcodec.

How to Run

mvn javafx:run

Planned Future Features

  • Additional effects: film grain, flicker, color fading.
  • Era presets: "20s Reel", "50s Film", "80s VHS".
  • Soundtrack processing (vintage audio effects).
